# UI 对比分析：AttaOS vs OpenClaw vs ZeroClaw

## 1. 技术栈对比

| 维度 | AttaOS | OpenClaw | ZeroClaw |
|------|--------|----------|----------|
| **框架** | Vue 3 + Pinia | Lit 3 (Web Components) | React 19 |
| **构建** | Vite 5 | Vite 7 | Vite 6 |
| **样式** | CSS Variables + 手写 CSS | CSS Variables + 手写 CSS | Tailwind CSS 4 |
| **路由** | Vue Router 4 | 自研 URL Router | React Router 7 |
| **状态管理** | Pinia (6 stores) | Lit Context + Signals | React Context |
| **语言** | TypeScript | TypeScript | TypeScript |
| **图标** | 无图标库 | 自定义 SVG | Lucide React |
| **桌面壳** | Tauri v2 (原生托盘+安装器+更新器) | 无 | 无 |
| **i18n** | 无 | 6 语言 (en/zh-CN/zh-TW/es/pt-BR/de) | 3 语言 (en/tr/zh) |

## 3. 页面功能对比

| **任务/会话** | 任务列表 + 详情 + 审批动作 | 会话管理 + 历史记录 | 无独立页面 |

## 7. AttaOS 的优势与差距

### 优势 (相比竞品)

1. **桌面原生体验** — Tauri 壳提供系统托盘、自动启动、安装向导、自动更新，竞品均无
2. **Flow 状态机可视化** — SVG 流程图 + YAML 查看，竞品均无 Flow 概念
3. **MCP 协议支持** — 独立 MCP 服务器管理页面，竞品未支持
4. **审批门控** — Task 粒度的 Approve/Deny/Request Changes，OpenClaw 仅有简单审批弹窗
5. **安全 E-Stop** — 系统托盘一键急停，竞品均无

### 差距 (需要补齐)

| 缺失功能 | OpenClaw 有 | ZeroClaw 有 | 优先级 |
|----------|:-----------:|:-----------:|:------:|
| **成本/用量分析** | 详细图表+CSV导出 | 分时段+模型分解 | 高 |
| **实时日志查看** | 级别过滤+轮询 | SSE 流+暂停/恢复 | 高 |
| **暗色模式** | 支持切换 | 默认暗色 | 中 |
| **定时任务 (Cron)** | 完整 CRUD+执行日志 | CRUD+状态追踪 | 中 |
| **多渠道集成** | 8 种 (Slack 等) | 集成配置 UI | 中 |
| **配置编辑** | 表单+JSON 双模式 | TOML 编辑器 | 中 |
| **系统诊断** | Debug 工具 | Doctor 页面 | 低 |
| **i18n 国际化** | 6 语言 | 3 语言 | 低 |
| **记忆管理 UI** | 无 | 增删查 | 低 |
